Welcome to the Todo App API. Use the following endpoints to interact with the application.

Authentication

POST /api/v1/auth/register
Register a new user
POST /api/v1/auth/login
Login user and get token
GET /api/v1/auth/me
Get current logged in user (requires authentication)

Tasks

GET /api/v1/tasks
Get all tasks for the authenticated user
POST /api/v1/tasks
Create a new task
GET /api/v1/tasks/:id
Get a single task by ID
PUT /api/v1/tasks/:id
Update a task
DELETE /api/v1/tasks/:id
Delete a task
PUT /api/v1/tasks/:id/toggle
Toggle task completion status

Getting Started

To get started, register a new user and use the returned JWT token to authenticate your requests.

Include the token in the Authorization header as: Bearer YOUR_TOKEN_HERE
